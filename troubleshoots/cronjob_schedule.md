Cron is an utility which automatically runs a certain script in a particular interval. To see which cronjobs are running you can use a commandline program called **crontab**. 

crontab -e : edits the current user's cron configuration file
crontab -l : lists all the cronjobs running under the current user
crontab -r : remove cronjobs running under the current user

In system administration Cron takes a valueable place like for automated backups, script execution, reminders or anything you want.
